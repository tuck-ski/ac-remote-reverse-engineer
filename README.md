# Reverse engineering the Daikin ARC478A5 AC remote control IR protocol

This repository contains Python scripts that I used to reverse engineer the ARC478A5 air conditioner remote control IR protocol, and to encode my own messages for the protocol that can be sent out from an IR blaster. Forked from [Spanfile/ac-remote-reverse-engineer](https://github.com/Spanfile/ac-remote-reverse-engineer). Please check the original repo for more information.

[More information in Spanfile's blog post.](https://blog.spans.fi/2024/04/16/reverse-engineering-the-yk-h531e-ac-remote-control-ir-protocol.html)
